# 權限


## 變更密碼

***變更指定使用者密碼***

```shell
sudo passwd USERNAME
```

***變更目前使用者密碼***

```shell
sudo passwd
```

***變更自己帳號密碼***

```shell
passwd
```

## 刪除使用者

```shell
sudo deluser USERNAME
sudo deluser --remove-home USERNAME
```
